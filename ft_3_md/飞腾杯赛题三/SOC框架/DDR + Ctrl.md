# DDR + Ctrl

## 作用
`DDR + Ctrl` 表示外部 DDR 存储器及其控制器（如 MIG/EMIF）。

它是 SoC 的主存，保存模型权重、输入特征图、输出结果和中间数据。

## 模块关系

```mermaid
flowchart LR
    DMA[AXI DMA] -->|AXI Burst R/W| DDR[(DDR + Ctrl)]
    NPU[NPU(可选AXI主口)] -->|AXI Burst R/W| DDR
```

## 存储规划建议
- `0x8000_0000` 起：输入特征图区
- `0x8100_0000` 起：权重区
- `0x8200_0000` 起：输出结果区
- `0x8300_0000` 起：中间缓冲区

## 带宽优化建议
- 尽量使用递增地址 burst 访问。
- 地址按总线宽度对齐（如 64/128bit 对齐）。
- 读写分块（tile）与 NPU 片上缓存大小匹配。
- 减少小包随机访问，降低行切换损耗。

## 验证要点
- burst 边界与长度约束满足 DDR 控制器规范。
- 同时存在读写时，数据一致性与吞吐率达标。
- 长时间压力仿真无超时、无错误响应。
